# Savoy Capital Fund

Landing page for Savoy Capital Fund, built with Next.js, TypeScript, and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm run start
```

## Deployment

This app deploys to **Railway** via GitHub auto-deploy from the `main` branch.

Railway detects the Next.js app and runs:
1. `npm install` (install dependencies)
2. `npm run build` (build the production app)
3. `npm run start` (start the Node server)

**Live URL:** _[Add Railway URL after first deploy]_

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- TypeScript
- Tailwind CSS v4
